Курсовая работа на тему "Восстановление параметров потока обращений к устройству хранения данных".
Решение содержит два проекта:
1. calcparam - является dll, написан на с++, осуществляет все расчеты
2. gui - графический интерфейс, вызывающий функции dll. Написан на C#
Исходный репозиторий - Nesterov. Новый репозиторий создан в момент появления gui. Содержит .gitignore
